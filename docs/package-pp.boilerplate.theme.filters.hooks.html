Package pp\boilerplate-theme\filters+hooks

Functions summary

pp_add_fresco_class Do some regex to add a fresco-image class to linked images, and open them with fresco.
pp_add_post_classes Expand post classes.
pp_body_classes

Expands the body classes added by WordPress.
Only called by add_filter('body_class','pp_body_classes');
pp_custom_login_logo Customize the login page to match site branding, link to homepage instead of wordpress.org
pp_custom_login_title Customize the login title
pp_custom_login_url Change the url when clicking login logo
pp_custom_post_type_archive_where

This allows you to pass a post_type argument to wp_get_archives as a string or an array.
pp_enqueue_jquery

Enqueue jQuery from Google CDN with fallback to local WordPress First we have a really complex method of loading jQuery from the CDN or local should the CDN fail. The code is adapted from https://gist.github.com/wpsmith/4083811. Then we load basic styles.
pp_enqueue_styles Use wp_enqueue to add theme stylesheet to wp_head()
pp_gallery_shortcode

Rewrite the markup for the gallery shortcode so that all gallery images can open with fresco.js. The link to attachment page is not respected, and size/column classes are added.
pp_load_fresco Load fresco.js and fresco.css
pp_register_menus Core menu registration.
pp_register_sidebars Register WordPress Sitebars
pp_stylesheet_uri Change the stylesheet url to our compiled stylesheet from Sassyplayte
pp_theme_setup Basic theme setup stuff like theme support
pp_wp_nav_menu_objects_sub_menu

Add a sub_menu option to wp_nav_menu(). Great for sidebar sub nav. Several other options are also added
