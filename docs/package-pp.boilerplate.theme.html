Functions summary

pp_acf_image Echos the output from pp_get_acf_image, which returns html for an ACF image
pp_custom_excerpt This function gets called in the template.
pp_document_title Print the document title, used in the title tag and og:title.
pp_excerpt Generates the excerpt portion of what gets printed in the template
pp_get_acf_image Returns html for an ACF image
pp_get_custom_fields Gets all of the custom text fields
pp_get_document_title Return document title, used in the title tag and og:title.
pp_get_page_title

Page title used in the content area of each page. Note this is kinda legacy. Could be replaced with wp_title().
pp_get_post_thumbnail_url

Returns the url of a post thumbnail for the current post. Can also specify post ID and image size.
pp_is_home_listing

Determines if the current listing is the homepage when the homepage is used for a feed of lates posts
pp_is_listing

Determines if a certain page is for a listing (search results, archive, homepage feed, etc) This is useful for creating a single listing UI module. WordPress's core tests fragment this between archives and search results.
pp_page_title Prints the content title generated by get_page_title
pp_post_image_classes Print classes depending on images that might be in a post
pp_sidebar_classes Add classes for the active sidebars
